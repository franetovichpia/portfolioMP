# sections/

Aquí vivirán las secciones de página (Hero, Services, Projects, About, Contact, etc.).

Cada sección se desarrollará en un paso posterior. Por ahora esta carpeta
está vacía intencionalmente — solo contiene esta guía de convención:

- Un archivo por sección: `hero.tsx`, `services.tsx`, `projects.tsx`, etc.
- Las secciones se componen dentro de `app/page.tsx`.
- La lógica de datos vive en `data/`, los tipos en `types/`.
